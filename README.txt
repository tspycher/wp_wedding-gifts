This Plugin is to create a simple wedding gift registery where vistor chan register them to donate gifts.

== Description ==

In this reduced version user need DB Access to add/edit gifts.

== Installation ==

1. Upload `wedding-gifts.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place the shortcode `[gifts]` somewhere in your site.
